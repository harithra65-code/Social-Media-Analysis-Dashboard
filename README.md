 Power BI Dashboard – Social Media Ads Analysis
 Overview:
This Power BI dashboard provides an interactive and comprehensive analysis of the Social Media Ads dataset. The report transforms raw data into meaningful insights through visualizations, KPI cards, and dynamic filtering options. It is designed to help users monitor business performance, analyze trends, compare categories, and support data-driven decision-making.
Key Features:
- Interactive dashboard with slicers and drill-down functionality.
- KPI cards displaying Revenue, Profit, Quantity Sold, Discount, Rating, Status, and Region.
- Column chart for revenue comparison across product categories.
- 100% stacked bar chart showing regional contribution by category.
- Area chart for quantity trends over time.
- Donut chart for order status distribution.
- Line and stacked column combo chart for comparing quantity sold and profit.
- Gauge visual for profit target tracking.
- Decomposition tree for detailed revenue analysis by Status, Region, and State.
Dashboard Insights:
- Analyze total revenue and profit performance.
- Identify high-performing product categories and regions.
- Monitor sales quantity trends across different time periods.
- Evaluate discount impact and customer rating patterns.
- Understand order status distribution.
- Drill down into revenue contributors for deeper analysis.

Tools and Technologies:
- Power BI Desktop
- Data modeling and relationships
- DAX measures and calculated fields
- Interactive charts and visualizations
- Slicers and filters
Usage:
1. Open the .pbix file in Power BI Desktop.
2. Explore the dashboard using slicers and filters.
3. Interact with charts to drill down into detailed insights.
4. Use the KPI cards and visuals to monitor business performance.
Conclusion:
This dashboard serves as a user-friendly business intelligence solution that converts advertising and sales data into actionable insights. It helps stakeholders track performance, identify trends, evaluate profitability, and make informed strategic decisions through an engaging and interactive Power BI interface.
